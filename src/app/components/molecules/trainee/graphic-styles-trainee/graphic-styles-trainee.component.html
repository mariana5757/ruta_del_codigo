🎨 Estilos Gráficos en UI

Aprenderás cómo estructurar y adaptar visualmente tus interfaces usando sistemas de rejilla, frameworks como Bootstrap o Material Design, y propiedades CSS responsive.

🔲 Sistemas de Rejilla en UI

Una rejilla permite distribuir los elementos visuales de manera ordenada y flexible en cualquier pantalla.

🟦 Bootstrap

Bootstrap es un framework frontend creado por Twitter para diseñar sitios web responsivos con componentes ya estilizados.

  • Sistema de grillas: Usa 12 columnas para crear layouts adaptables.
  • Componentes: Botones, formularios, alertas, modales, tarjetas, etc.
  • Utilidades CSS: Clases para márgenes, padding, alineación, colores, etc.
  • JavaScript integrado: Carruseles, menús, tooltips, etc.
  • Personalización: Puedes modificar variables Sass o sobrescribir estilos.

🟩 Material Design

Material Design es un sistema de diseño creado por Google. Tiene una librería visual coherente basada en HTML y CSS con principios como elevación, animaciones, y componentes con enfoque en la usabilidad.

📱 Propiedades CSS para diseño responsive

Estas propiedades permiten que el diseño se adapte a múltiples tamaños de pantalla:

  • width / height
  • max-width / min-width
  • rem, em, vw, vh
  • flexbox y grid
  • object-fit / object-position
  • @media queries para aplicar estilos según el ancho de pantalla
