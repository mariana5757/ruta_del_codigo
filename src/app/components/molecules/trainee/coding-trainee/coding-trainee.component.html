🔍 Reto 1: Sumar Números Pares

Tienes una lista de números. El objetivo es sumar solo los números pares. Observa el siguiente código y corrige cualquier error que encuentres. 


function sumaPares(numeros) {
  let suma = 0;
  for (let i = 0; i < numeros.length; i++) {
    if (numeros[i] % 2 === 0) {
      suma += numeros[i];
    }
  }
  return suma;
}

// Prueba
console.log(sumaPares([1, 2, 3, 4, 5])); // Debería mostrar 6

Tu tarea: Revisa la lógica del código. ¿Está bien planteado? ¿Hay errores de sintaxis o lógica? Intenta escribir una versión optimizada.

🧪 Reto 2: Verificar Palíndromos

Un palíndromo es una palabra que se lee igual al derecho y al revés, como "oso" o "reconocer". Analiza la siguiente función e identifica qué podrías mejorar o corregir: 


function esPalindromo(texto) {
  const limpio = texto.toLowerCase().replace(/[^a-z0-9]/g, '');
  const invertido = limpio.split('').reverse().join('');
  return limpio === invertido;
}

// Prueba
console.log(esPalindromo("Anita lava la tina")); // true

Tu tarea: Asegúrate de que funcione con frases, con o sin espacios, y sin importar mayúsculas o signos.

🐞 Reto 3: Depura este Código

El siguiente código tiene un error que impide que se sume correctamente. ¿Puedes encontrarlo y corregirlo? 


function sumarLista(lista) {
let total = 0;
lista.forEach(function(numero) {
    total += numero;
})
return total;
